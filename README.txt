Files used in this assignment
1. DocWordCount.java
2. TermFrequency.java
3. TFIDF.java
4. Search.java
5. Cantrbry corpus with 8 documents.

Place all the files in a folder and name it as Assignment2

--------------------Execution Instructions for DocWordCount------------------------

1) Create a directory for assignment2 using the following command
	$ hadoop fs -mkdir /user/cloudera/assignment2/
2) Create a directory for input files directories
    	$ hadoop fs -mkdir /user/cloudera/assignment2/input

3) Add all the 8 input files to the directory created in the previous step.
	$ hadoop fs -put cantrbry/* /user/cloudera/assignment2/input

4) Compile the DocWordCount.java file  and create a jar using the below commands

	$ mkdir -p build
	$ javac -cp /usr/lib/hadoop/*:/usr/lib/hadoop-mapreduce/* DocWordCount.java -d build -Xlint 
	$ jar -cvf docWordCount.jar -C build/ .

5) Use the following command to run the docWordCount.jar on the input files to calculate the Document Word Count.
	$ hadoop jar docWordCount.jar org.myorg.DocWordCount /user/cloudera/assignment2/input /user/cloudera/assignment/docwordcount/output

6) Output generated by the DocWordCount can be accessed using the following commands.

	//read the output
	$ hadoop fs -cat /user/cloudera/assignment2/docwordcount/output/*
	//copy output to local
	$ hadoop fs -copyToLocal /user/cloudera/assignment2/docwordcount/output/* .

7) For deleting the output directory, run the below command:
    	$ hadoop fs -rm -r /user/cloudera/assignment2/docwordcount/output
